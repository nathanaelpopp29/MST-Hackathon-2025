# MST-Hackathon-2025
This repository contains the files for two games developed during the Missouri University of Science and Technology 2025 Hackathon. The theme of the hackathon was “Games,” and each team was tasked with creating a game of their choice using any tools or technologies they preferred. Since my teammates and I had little experience developing games beyond backend or terminal-based projects, we first needed to identify a suitable game engine. After researching several options, we decided to use the open-source game engine Godot.

# Godot Beginner Tutorial - Baseline 2D Side Scroller
This folder contains a simple 2D side-scroller built by following a Godot tutorial on YouTube. The project was used to familiarize our team with the Godot engine and to generate help ideas for the game we planned to develop for the event.

# Blob And Friends
This folder contains our group’s game submission for the event. Blob and Friends is a 2D top-down game in which the player explores an enclosed environment to locate specific stationary entities while avoiding enemies. Since this was our first experience developing a project of this kind and the time available was limited, the game remains relatively simple and focuses on implementing the concepts introduced in the tutorial.
