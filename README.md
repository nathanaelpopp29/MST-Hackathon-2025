# MST-Hackathon-2025
This repository contains the files for two separate games made during the Missouri University of Science and Technology 2025 Hackathon. This hackathon's theme was 'games' and each team was tasked with building a game of their choice in whatever way they wanted. As my team members and I had little to no experience in making games outside of backend/terminal based ones, we had to find an open source game engine we could use. After some investigating, we settled on using the game engine known as Godot.

# Godot Beginner Tutorial - Baseline 2D Side Scroller
This folder contains a baseline 2D side scroller game that was created by following along with a youtube tutorial for Godot. The creation of this game was used to familiarize our group with how Godot worked and brainstorm ideas for the game we would make for the event.

# Blob And Friends
This folder contains our groups game submission for the event. Blob and Friends is a 2D top-down game where you traverse an enclosed area and attempt to find certain non-moving entities while avoiding enemies. As this was the first time any of us had done something like this and the time limit was short, the game is very simple.
